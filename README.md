Sou graduando em **Ciência da Computação pela Universidade Estadual de Santa Cruz (UESC)**, com experiência prática consolidada através de duas residências de desenvolvimento de software:

- 🐍 **Residência Conecta e Capacita** - Desenvolvedor Back-End com Python

Implementação acadêmica do jogo de Go aplicando conceitos de **sistemas distribuídos** para comunicação entre máquinas. Explora sincronização de estado e arquitetura cliente-servidor.

Aplicação **cliente-servidor com sockets TCP** e interface gráfica em **Tkinter**. Demonstra comunicação em rede e programação orientada a eventos na interface.

### 🖥️ [Simulador de CPU (OAC)](https://github.com/gilbertcm/OAC-Project)
Simulador de CPU com execução de instruções **Assembly**, desenvolvido para a disciplina de Organização e Arquitetura de Computadores. Trabalha conceitos de baixo nível e ciclo de instrução.
